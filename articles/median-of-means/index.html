Median-of-Means Estimator

A technical note on why ordinary means can become unstable under contaminated noise, and how median-of-means reduces estimation tail risk.

2026-06-30 8 min read Quarto-style note

A tiny example / 一个极简例子

Suppose we want to estimate the true average level of a dataset. Most observations are near 10, but one record is 1000.

pythoninput
x = [10, 11, 9, 10, 12, 11, 10, 9, 10, 1000]
(10 + 11 + 9 + 10 + 12 + 11 + 10 + 9 + 10 + 1000) / 10 = 109.2

这个结果明显被 1000 拉歪了。真实中心其实更接近 10。

Key intuition: ordinary mean asks, "What is the average of all points?" Median-of-means asks, "What is the typical average among many small groups?"

Notebook experiment / 重复随机实验

上面的例子只是一个手工构造的异常值案例。为了验证 Median-of-Means 是否真的能在随机数据中带来更稳定的估计，我们做一个 Monte Carlo 实验：每次生成 200 个样本，其中 95% 来自正常噪声，5% 来自极端噪声。真实中心设为 10，重复 5000 次。

pythonIn [1]
import numpy as np
import pandas as pd
import matplotlib.pyplot as plt

rng = np.random.default_rng(42)

def median_of_means(x, k=20):
    x = np.array(x, copy=True)
    rng.shuffle(x)
    blocks = np.array_split(x, k)
    block_means = [block.mean() for block in blocks]
    return np.median(block_means)

def generate_sample(n=200, true_center=10, contamination_rate=0.05):
    x = rng.normal(true_center, 1, size=n)
    mask = rng.random(n) < contamination_rate
    x[mask] = rng.normal(true_center, 50, size=mask.sum())
    return x

def run_experiment(repeats=5000, n=200, true_center=10):
    records = []
    for _ in range(repeats):
        x = generate_sample(n=n, true_center=true_center)
        records.append({
            "sample_mean": np.mean(x),
            "median": np.median(x),
            "median_of_means": median_of_means(x, k=20),
        })
    return pd.DataFrame(records)

results = run_experiment()

Interpretation: sample mean is not wrong every time, but it occasionally goes very wrong. Median-of-means is not claiming to have the smallest variance in every clean setting; its value is reducing tail risk of estimation error under heavy-tailed or contaminated noise.

Takeaway

Median-of-Means 降低的是重尾噪声下的估计尾部风险，而不只是普通意义上的方差。

This matters in salary data, abnormal transactions, long-tail response times, and noisy scraped records: first ask what assumptions make the mean stable.
